## ✨ About Altair

Altair is an open-source project management platform specifically designed for
ADHD brains. Named after the brightest star in the Aquila constellation—a
beacon for navigation—Altair helps you stay oriented and make progress even when
everything feels chaotic.

### Why Altair?

Traditional project management tools fail ADHD users because they're built for
neurotypical thinking patterns. Altair is different:

- **🎯 Instant Capture** - Save tasks before they vanish from working memory
- **⏱️ Time Awareness** - Visual representations designed for time blindness
- **🧩 Smart Breakdown** - AI-powered decomposition into manageable chunks
- **🎮 Gentle Gamification** - Progress tracking without exploitative mechanics
- **📝 Automatic Documentation** - Capture insights without interrupting flow
- **🔒 Privacy First** - Your data stays yours, always

## 🛠️ Tech Stack

- **Backend:** FastAPI (Python)
- **Frontend:** Flutter (web + mobile)
- **Database:** PostgreSQL
- **Architecture:** Modular design with offline-first capabilities

### Why This Stack?

- **FastAPI:** Modern, fast, with excellent async support and automatic API documentation
- **Flutter:** Single codebase for web and mobile,
  smooth animations perfect for ADHD-friendly UI
- **PostgreSQL:** Rock-solid reliability with proven scalability and robust
  open-source ecosystem
- **Open Source First:** Primary focus on self-hosted, open-source deployment
  with optional managed services for less technical users planned

## 🌟 Philosophy

Altair is built on these core principles:

1. **Neurodiversity-affirming** - ADHD is a different operating system, not a deficiency
2. **Privacy by design** - Your data, your control
3. **Open source forever** - Community-owned and transparent
4. **Evidence-based** - Grounded in ADHD research and lived experience
5. **Accessible first** - WCAG compliance + ADHD-specific accommodations
6. **Dogfooding** - We use Altair to build Altair

## 🗺️ Development Approach

### Build in Public

We develop Altair in the open, sharing our progress, challenges, and learnings.
The main developer uses Altair to manage Altair's development—this dogfooding
ensures features actually work for ADHD brains.

### Deployment Philosophy

- **Primary:** Self-hosted, open-source deployment
- **Alternative:** Free-tier and paid managed services (coming in future releases)
- **Goal:** Make it accessible to technical and non-technical users alike
