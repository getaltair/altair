## ✨ About Altair

Altair is an open-source project management platform specifically designed for ADHD brains. Named after the brightest star in the Aquila constellation—a beacon for navigation—Altair helps you stay oriented and make progress even when everything feels chaotic.

### Why Altair?

Traditional project management tools fail ADHD users because they're built for neurotypical thinking patterns. Altair is different:

- **🎯 Instant Capture** - Save tasks before they vanish from working memory
- **⏱️ Time Awareness** - Visual representations designed for time blindness
- **🧩 Smart Breakdown** - AI-powered decomposition into manageable chunks
- **🎮 Gentle Gamification** - Progress tracking without exploitative mechanics
- **📝 Automatic Documentation** - Capture insights without interrupting flow
- **🔒 Privacy First** - Your data stays yours, always

## 🚀 Current Status

**Pre-Alpha Development** - Not ready for use yet, but you can follow along!

We're building in public and dogfooding from day one. Check out our [Project Roadmap](docs/roadmap.md) to see what we're working on.

## 🛠️ Tech Stack

- **Backend:** Go with Echo framework
- **Frontend:** Flutter (web + mobile)
- **Database:** SurrealDB 2.0 (main) + PostgreSQL (auth/billing)
- **Architecture:** Modular monolith with event-driven patterns

## 🌟 Philosophy

Altair is built on these core principles:

1. **Neurodiversity-affirming** - ADHD is a different operating system, not a deficiency
2. **Privacy by design** - Your data, your control
3. **Open source forever** - Community-owned and transparent
4. **Evidence-based** - Grounded in ADHD research and lived experience
5. **Accessible first** - WCAG compliance + ADHD-specific accommodations
