CSS Implementation

/* Neo-Brutalist Diamond Logo */
.diamond-logo {
    display: inline-block;
    width: 40px;
    height: 40px;
    transform: rotate(45deg);
    border: 3px solid #000;
    background: #FFD93D;
    position: relative;
}

/* Light facet */
.diamond-logo::before {
    content: '';
    position: absolute;
    top: 0; left: 0;
    width: 50%; height: 50%;
    background: rgba(255, 255, 255, 0.4);
}

/* Dark facet */
.diamond-logo::after {
    content: '';
    position: absolute;
    bottom: 0; right: 0;
    width: 50%; height: 50%;
    background: rgba(0, 0, 0, 0.1);
}

Usage Guidelines

  • DO: Use thick borders (3px minimum)
  • DO: Apply hover effects with transform and shadow
  • DO: Use flat colors - no gradients
  • DON'T: Use gradients or soft shadows
  • DON'T: Make borders thinner than 2px
